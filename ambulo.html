Ambulo

Ambulo is a hiking web-based application that uses Trail API for locations and Flickr API for images of trails. Users are able to search by city and view local trails.

Search Bar Autocomplete

The autocomplete suggestions help prevent errors and streamline the search process.

Trail Images

The Flickr images will help hikers decide which trails they might want to checkout in the future.

Selecting a specific trail

When a user clicks on a trail name, they can get instant information such as location and information about activities provided by the Trail API. In the future, I would like to implement a weather API to help users plan their trips.

Favoriting a Trail

After logging in, users can save the trails they've liked by adding each trail to their Favorites page. Clicking on the "View larger map" bar will take the users to a Google Maps page where users can find directions.

Login System

I developed and design the login system using React & Firebase.
